
# שינוי מבנה המשימות למעקב יומי (Daily Habit Tracking)

## הבנת הבעיה

המערכת הנוכחית מתייחסת למשימות כ-**checklist חד-פעמי**:
- "שגרת בוקר מובנית" - וי אחד ונגמר
- "הפסקת שימוש בקנאביס" - וי אחד ונגמר

**מה שאתה צריך**: מעקב יומי חוזר (Habit Tracking) שבו:
- כל יום מתאפסות המשימות
- אורורה יכולה לסמן דרך השיחה "עשית X היום? יופי! ✅"
- בסוף היום/שבוע רואים סיכום: "השבוע הצלחת 5/7 ימים בשגרת בוקר!"

---

## תוכנית טכנית

### שלב 1: טבלה חדשה למעקב יומי

יצירת טבלת `daily_habit_logs` שתעקוב אחרי כל יום בנפרד:

```text
daily_habit_logs
├── id (uuid)
├── user_id (uuid)
├── habit_item_id (uuid) → מקושר ל-aurora_checklist_items
├── track_date (date) → התאריך הספציפי
├── is_completed (boolean) → האם הושלם באותו יום
├── completed_at (timestamp)
├── completed_by ('manual' | 'aurora') → מי סימן
└── notes (text) → הערות אופציונליות
```

### שלב 2: הוספת שדה `is_recurring` לפריטי checklist

עדכון `aurora_checklist_items`:
```sql
ALTER TABLE aurora_checklist_items 
ADD COLUMN is_recurring boolean DEFAULT false;
```

משימות מתוך "הרגלים לבנות" ו"הרגלים להפסיק" יסומנו כ-recurring.

### שלב 3: עדכון ה-UI

**ChecklistsCard.tsx** ישתנה להציג:
- משימות חד-פעמיות: כמו היום (וי אחד)
- משימות יומיות חוזרות: 
  - רצף של ימים (streak): "🔥 5 ימים רצוף!"
  - מעקב היום: האם עשית היום?
  - היסטוריה: לוח שבועי/חודשי עם וי לכל יום

### שלב 4: עדכון Aurora

**aurora-chat/index.ts** יקבל מידע על:
- מצב היום הנוכחי לכל הרגל
- רצפים (streaks)
- מה עוד לא דווח היום

דוגמה:
```
## מעקב הרגלים יומי (היום: 31/01/2026)
- שגרת בוקר: ❓ טרם דווח (streak: 3 ימים)
- פעילות גופנית: ❓ טרם דווח (streak: 0)
- הימנעות מקנאביס: ✅ הושלם! (streak: 5 ימים 🔥)
```

כשמשתמש אומר "עשיתי אימון היום", אורורה תזהה ותסמן:
```
[habit:complete:פעילות גופנית יומית]
```

### שלב 5: תגית חדשה לאורורה

הוספת תגיות חדשות:
- `[habit:complete:שם_הרגל]` - סימון הרגל כהושלם להיום
- `[habit:reset]` - איפוס יומי (קורה אוטומטית בחצות)

| `src/components/dashboard/unified/ChecklistsCard.tsx` | UI למעקב יומי עם streak |

| `src/hooks/aurora/useAuroraChat.tsx` | עיבוד תגית habit:complete |

## איך אורורה תתפקד

כשתפתח שיחה עם אורורה, היא תראה:
> "בוקר טוב! 👋 איך היה לילה? ראיתי שכבר יום חמישי ברצף בלי קנאביס - מדהים! 🔥
> 
> מה עם שגרת הבוקר והאימון היום?"

כשתגיד "עשיתי אימון ושגרת בוקר", היא תגיב:
> "מעולה! ✅ סימנתי את שניהם 🎉
> 
> [habit:complete:שגרת בוקר מובנית]
> [habit:complete:פעילות גופנית יומית]
> 
> זה כבר 4 ימים רצוף עם שגרת בוקר - keep going! 💪"

## תוצאה צפויה

1. ✅ המשימות יהפכו למעקב יומי חוזר
2. ✅ כל יום מתאפסות המשימות
3. ✅ אורורה תוכל לסמן הרגלים דרך השיחה
4. ✅ תראה streak (רצף ימים) לכל הרגל
5. ✅ סיכום שבועי/חודשי של ההתקדמות
