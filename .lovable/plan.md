

# תוכנית עיצוב מחדש של דף הפרופיל (ProfileDrawer)

## הבנת הבעיה
הפרופיל הנוכחי (`ProfileDrawer.tsx`) מציג:
- טאבים של Profile / Focus / AI Analysis
- תצוגה גולמית של נתוני השאלון
- תצוגה כפולה של מידע שכבר קיים בדאשבורד (AI Analysis)
- עיצוב בסיסי ולא מרגש

## הרעיון החדש: "כרטיס זהות דיגיטלי" (Digital Identity Card)

במקום פרופיל משעמם, ניצור **כרטיס זהות אישי** שמסכם את כל מה שהמערכת יודעת על המשתמש בצורה ויזואלית מרהיבה - כמו דף דמות ב-RPG.

## שינויים טכניים

### קבצים לעריכה:
1. **`src/components/dashboard/ProfileDrawer.tsx`** - שכתוב מלא

### מקורות נתונים:
- `useUnifiedDashboard()` - Level, XP, Tokens, Streak, Identity Title, Values, Principles, etc.
- `useLaunchpadSummary()` - Consciousness score, Career path, Behavioral insights
- `useLaunchpadData()` - First week habits data
- `useMultiThreadOrbProfile()` - For the Orb

### רכיבים חדשים (בתוך ProfileDrawer):
1. `ProfileHeroSection` - Orb + Identity Title + Stats
2. `ProfileScoresBar` - Consciousness/Clarity/Readiness
3. `ProfileValuesSection` - Values hierarchy + Traits
4. `ProfileDirectionCard` - Life direction with clarity
5. `ProfileCareerCard` - Current status & aspiration
6. `ProfileTransformationCard` - Habits to change/build
7. `ProfileActions` - Edit Journey + Regenerate buttons

### עיצוב:
- שימוש ב-Glassmorphism (`backdrop-blur-xl`, `bg-card/60`)
- גרדיאנטים תואמים לצבעי ה-Theme
- אנימציות עדינות עם Framer Motion
- עיצוב Game UI עם גבולות זוהרים

## יתרונות הגישה החדשה

1. **אין כפילויות** - מידע ייחודי שלא קיים בדאשבורד
2. **מיקוד בזהות** - מראה למשתמש "מי אתה" במבט אחד
3. **Game Feel** - מרגיש כמו דף דמות ב-RPG
4. **מהירות** - הכל במסך אחד בלי טאבים
5. **תחושת ערך** - המשתמש מרגיש שהמערכת מכירה אותו
