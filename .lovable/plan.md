
# תוכנית אסטרטגית: דף בית ממוקד התמכרות + חוויית משחק

## מחקר ומימצאים

בהתבסס על מחקר פסיכולוגי מבוסס על Dopamine Loops:

| מנגנון | השפעה | איך ליישם |
|--------|--------|-----------|
| **Variable Rewards** | שחרור דופמין גבוה יותר מתגמולים אקראיים | הפתעות אקראיות, אייקוני Ego States מסתובבים |
| **Progress Visualization** | מוטיבציה לסיום | Bar מתקדם, אחוז השלמה, רמות ברורות |
| **Loss Aversion** | פחד לאבד > רצון לקבל | Streak counter, "אל תאבד את ה-7 ימים שלך" |
| **Social Proof** | אמון + FOMO | "1,234 משתמשים פעילים עכשיו" |
| **Commitment Loop** | השקעה קטנה → מחויבות גדולה | Launchpad: 10 שלבים קטנים |

## חלק 3: שיפורי Gamification - מנגנוני התמכרות

### 1. Progress Bar מרכזי בכל סקשן

במקום רק כרטיסיות, נוסיף visual progress:

```text
┌────────────────────────────────────────────────────────────────────┐
│  ⭐ Level 7                                              1,450 XP   │
│  ████████████░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░░   48% to LVL 8 │
│                                                                      │
│  🔥 Streak: 12 Days    💰 Tokens: 340    🏆 15/50 Achievements     │
└────────────────────────────────────────────────────────────────────┘
```

### 2. Variable Rewards - הפתעות

- Ego State Icons שמסתובבים סביב Aurora באופן אקראי
- "Daily Bonus" התראה פוטנציאלית
- Achievement pop-ups עם אנימציות

### 3. Loss Aversion

הוספת טקסטים כמו:
- "🔥 12 ימים רצופים - אל תאבד את הסטריק!"
- "💎 עוד 50 XP ואתה מגיע לרמה 8!"
- "⏰ יש לך משימה שמחכה - אל תאבד את ה-XP"

### 4. Social Proof (למסרים)

- "1,234 משתמשים התחברו השבוע"
- "78 משתמשים עלו רמה היום"
