
# תוכנית: התאמת סשן ההיפנוזה לפרופיל ולשבוע הנוכחי

## סקירה
כרגע מודל ההיפנוזה מאפשר למשתמש לבחור מטרות קבועות מראש (רגיעה, מיקוד, אנרגיה, שינה) או להזין מטרה ידנית. המערכת האחורית (Edge Function) כבר טוענת את הנתונים האישיים של המשתמש, אבל **הממשק לא מציג את המטרה המותאמת אישית כברירת מחדל**.

## שינויים נדרשים

### 1. שילוב hook הפרופיל והשבוע הנוכחי
**קובץ:** `src/components/dashboard/HypnosisModal.tsx`

- ייבוא ושימוש ב-`useDailyHypnosis` שכבר קיים
- טעינת המטרה המוצעת אוטומטית לשדה הקלט
- הצגת הקשר נוסף (שבוע נוכחי, אבן דרך) בממשק

### 2. שיפור ממשק ההגדרות
- הוספת כרטיס ראשי המציג "מטרת השבוע" הנגזרת מתוכנית ה-90 ימים
- החלפת כפתורי הפריסטים הכלליים בכפתור "המטרה שלי לשבוע" בולט
- הצגת תיאור קצר של אבן הדרך הנוכחית

## פרטים טכניים

### שינויים בקובץ HypnosisModal.tsx

**ייבואים חדשים:**
```typescript
import { useDailyHypnosis } from '@/hooks/useDailyHypnosis';
```

**שימוש ב-hook:**
```typescript
const { 
  currentMilestone, 
  launchpadSummary, 
  suggestedGoal, 
  isLoading: isLoadingContext 
} = useDailyHypnosis();
```

**אתחול המטרה אוטומטית:**
- כאשר ה-modal נפתח והנתונים נטענו, המטרה המוצעת תוזן אוטומטית לשדה
- הוספת `useEffect` שמעדכן את ה-goal כשה-`suggestedGoal` זמין

**רכיב חדש - כרטיס השבוע:**
- כרטיס מודגש בראש ה-setup screen
- מציג את מספר השבוע ושם אבן הדרך
- כפתור "השתמש במטרת השבוע" שמזין את המטרה

**שיפורים נוספים:**
- הפריסטים הקיימים נשארים כאופציה משנית
- מצב טעינה קצר בזמן שה-context נטען
- תמיכה בשני הכיוונים (RTL/LTR)

### לוגיקת העדיפות למטרה (כפי שהיא ב-useDailyHypnosis)
1. **עדיפות ראשונה:** כותרת אבן הדרך הנוכחית מתוכנית ה-90 ימים
2. **עדיפות שנייה:** השאיפה המרכזית מכיוון החיים
3. **עדיפות שלישית:** תחום הצמיחה הראשון
4. **עדיפות רביעית:** מצב התודעה הנוכחי
5. **ברירת מחדל:** "התפתחות אישית והתעלות"

## השפעה
- **חוויית משתמש:** הסשן מרגיש מותאם אישית מהרגע הראשון
- **הקשר:** המשתמש רואה את ההתקדמות שלו בתוכנית ה-90 ימים
- **פחות החלטות:** המטרה כבר מוכנה, פשוט ללחוץ "התחל"
- **גמישות:** עדיין אפשר לבחור מטרה אחרת אם רוצים
