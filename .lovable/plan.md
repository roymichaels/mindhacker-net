

# הרחבת רשימות ההרגלים - כיסוי מלא

## מה חסר כרגע

### HABITS_TO_QUIT (להפסיק) - חסרים:
| קטגוריה | חסר |
|---------|-----|
| חומרים | אלכוהול, סמים, עישון/ניקוטין, קפאין מוגזם, סוכר/מתוקים |
| התמכרויות | פורנו, הימורים, קניות כפייתיות, גיימינג מוגזם |
| התנהגויות | שקרים, כעסים לא מבוקרים, איחורים כרוניים, אי-הקשבה |
| פיננסי | הוצאות לא מחושבות, חובות, חיים מעבר ליכולת |
| חברתי | בידוד עצמי, תלות באחרים, רכילות |
| מנטלי | דאגנות יתר, פרפקציוניזם משתק, השוואה לאחרים |

### HABITS_TO_BUILD (לבנות) - חסרים:
| קטגוריה | חסר |
|---------|-----|
| בריאות | שתיית מים, תזונה מאוזנת, בדיקות רפואיות |
| פיננסי | מעקב הוצאות, חיסכון, השקעות |
| יחסים | זמן איכות עם משפחה, הקשבה פעילה, מתן מחמאות |
| התפתחות | לימוד שפה, קורסים מקצועיים, האזנה לפודקאסטים |
| פרודוקטיביות | תכנון שבועי, ביקורת יומית, אמירת "לא" |
| רוחני | הכרת תודה, התנדבות, חיבור לקהילה |

## רשימות מורחבות

### HABITS_TO_QUIT החדש (30+ פריטים):

**חומרים והתמכרויות:**
- 🍺 אלכוהול מוגזם
- 💊 סמים/שימוש בחומרים
- 🚬 עישון / ניקוטין
- ☕ קפאין מוגזם
- 🍫 סוכר והתמכרות למתוקים
- 🎰 הימורים
- 🔞 פורנו / תוכן מיני
- 🎮 גיימינג כפייתי

**התנהגויות מזיקות:**
- 📱 סקרולינג אינסופי ברשתות
- ⏰ דחיינות ודחיית משימות
- 📺 צפייה מוגזמת בנטפליקס/יוטיוב
- 🍕 אכילה רגשית / לא בריאה
- 🍔 אוכל זבל ושתייה מתוקה
- 🌙 שהייה ערים עד מאוחר
- ⌛ בזבוז זמן על דברים לא חשובים
- 🛒 קניות כפייתיות

**דפוסים מנטליים:**
- 🗣️ דיבור עצמי שלילי
- 😤 תלונות ושליליות
- 📊 השוואה מתמדת לאחרים
- 😰 דאגנות יתר
- 🎭 פרפקציוניזם משתק
- 💭 חשיבת יתר (overthinking)

**יחסים:**
- 👥 יחסים שמדכאים אותי
- 🤐 הימנעות מעימותים נחוצים
- 👂 אי-הקשבה / קטיעת אנשים
- 🗨️ רכילות ולשון הרע
- 🙈 בידוד עצמי מהעולם
- 🧲 תלות יתר באחרים

**פיננסי:**
- 💸 הוצאות לא מחושבות
- 💳 חיים מעבר ליכולת
- 📉 הימנעות מלהסתכל על המצב הכלכלי

**אחר:**
- ⏱️ איחורים כרוניים
- 🤥 שקרים קטנים
- 😡 התפרצויות כעס
- 🙅 אמירת כן לכל דבר

---

### HABITS_TO_BUILD החדש (30+ פריטים):

**שגרות ומבנה:**
- 🌅 שגרת בוקר מובנית
- 📋 תכנון יום/שבוע מראש
- 🎯 הגדרת מטרות יומיות
- ✅ סיום משימות עד הסוף
- 🚫 אמירת "לא" לדברים לא חשובים
- 📊 ביקורת יומית/שבועית

**גוף ובריאות:**
- 💪 פעילות גופנית יומית
- 💧 שתיית מים מספקת
- 🥗 תזונה מאוזנת ובריאה
- 😴 שינה איכותית ומספקת
- 🥶 מקלחת קרה / אתגר פיזי
- 🏥 בדיקות רפואיות תקופתיות

**למידה והתפתחות:**
- 📚 למידה יומית (30+ דקות)
- 📖 קריאת ספרים (לא רשתות)
- 🎧 האזנה לפודקאסטים מלמדים
- 🌍 לימוד שפה חדשה
- 🎓 קורסים מקצועיים

**עסקים וקריירה:**
- 🏗️ עבודה על הפרויקט/העסק שלי
- 🎯 תרגול מיומנות מקצועית
- 🤝 נטוורקינג ויצירת קשרים
- 📞 יצירת קשר עם לקוחות/שותפים
- 📝 תיעוד רעיונות ותובנות

**מנטלי ורגשי:**
- 🧘 מדיטציה / רפלקציה יומית
- ✍️ כתיבה יומית / ג'ורנלינג
- 🙏 תרגול הכרת תודה
- 🧠 עבודה על חסמים מנטליים
- 💆 ניהול סטרס בריא

**יחסים:**
- 👨‍👩‍👧 זמן איכות עם משפחה
- 💑 השקעה בזוגיות
- 👂 הקשבה פעילה לאחרים
- 💬 תקשורת כנה ופתוחה
- 🤲 עזרה לאחרים / התנדבות

**פיננסי:**
- 📒 מעקב הוצאות יומי
- 💰 חיסכון קבוע
- 📈 למידה על השקעות
- 💵 בניית מקור הכנסה נוסף
