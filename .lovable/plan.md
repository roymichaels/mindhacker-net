

## תכנית: הפרדת פאנל אדמין מפאנל מאמנים

### סקירת השינויים

נפריד את הפאנל לשלושה אזורים נפרדים:
- **פאנל אדמין** (`/panel`) - ניהול כללי של הפלטפורמה
- **פאנל מאמנים** (`/coach`) - ניהול הפרקטיקה האישית
- **פאנל שותפים** (`/affiliate`) - ניהול קישורים והפניות

```text
לפני:
┌─────────────────────────────────────┐
│         /panel (מאוחד)              │
│  ┌───────┐ ┌──────┐ ┌───────────┐  │
│  │ Admin │ │Coach │ │ Affiliate │  │
│  └───────┘ └──────┘ └───────────┘  │
└─────────────────────────────────────┘

אחרי:
┌────────────┐  ┌────────────┐  ┌────────────┐
│   /panel   │  │   /coach   │  │ /affiliate │
│   (Admin)  │  │  (מאמנים)   │  │  (שותפים)  │
└────────────┘  └────────────┘  └────────────┘
       ↑              ↑              ↑
       └──────────────┴──────────────┘
          כפתור מעבר מהיר בין פאנלים
```

### פרטים טכניים

**קבצים חדשים:**
- `src/components/panel/AdminSidebarNew.tsx` - סיידבר אדמין נקי
- `src/components/panel/CoachSidebar.tsx` - סיידבר מאמנים
- `src/components/panel/AffiliateSidebar.tsx` - סיידבר שותפים
- `src/components/panel/CoachPanel.tsx` - layout למאמנים
- `src/components/panel/AffiliatePanel.tsx` - layout לשותפים
- `src/components/panel/RoleSwitcher.tsx` - כפתור מעבר בין פאנלים
- `src/pages/panel/CoachTheme.tsx` - עיצוב פרופיל למאמנים

**קבצים למחיקה:**
- `src/components/admin/AdminSidebar.tsx` (הישן)
- `src/pages/AdminDashboard.tsx` (הישן)
- כל הנתיבים תחת `/admin/*`

**שינויים ב-App.tsx:**
- מחיקת כל ה-routes של `/admin/*`
- הוספת routes ל-`/coach/*`
- הוספת routes ל-`/affiliate/*`
- עדכון `/panel/*` להיות רק לאדמינים

**RoleRoute guards:**
- `/panel/*` - admin בלבד
- `/coach/*` - practitioner בלבד
- `/affiliate/*` - affiliate בלבד
