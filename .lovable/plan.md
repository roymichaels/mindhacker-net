

## תכנית: העברת כפתור הפילטר לתוך קונטיינר הפוסטים

### מה יבוצע

נעביר את רכיב `CategoryFilter` לתוך הקונטיינר של הפוסטים ונמחק את הקונטיינר הנפרד שלו.

### מצב נוכחי
```text
┌─────────────────────────────┐
│     Post Editor Box         │
└─────────────────────────────┘
┌─────────────────────────────┐
│  🔽 Filter Container (נפרד) │  ← למחוק
└─────────────────────────────┘
┌─────────────────────────────┐
│      Posts Container        │
│   ┌───────────────────┐     │
│   │   אין פוסטים עדיין   │     │
│   └───────────────────┘     │
└─────────────────────────────┘
```

### מצב חדש
```text
┌─────────────────────────────┐
│     Post Editor Box         │
└─────────────────────────────┘
┌─────────────────────────────┐
│  🔽 Filter (בתוך הקונטיינר)  │
│   ┌───────────────────┐     │
│   │   אין פוסטים עדיין   │     │
│   └───────────────────┘     │
└─────────────────────────────┘
```

---

### פרטים טכניים

**קובץ: `src/components/community/CommunityFeed.tsx`**

1. מחיקת הקונטיינר הנפרד של הפילטר (שורות 144-150)
2. הוספת `CategoryFilter` בתוך קונטיינר הפוסטים, לפני תוכן הפוסטים
3. שימוש במבנה אחד עם `rounded-xl` שמכיל גם את הפילטר וגם את הפוסטים
